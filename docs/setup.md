# 你需要的配備

## 開始前，你需要準備哪些東西？

1. 你最好最好有一台 Mac 電腦。
2. Google 帳號，我們會使用 [Google Colab](https://colab.research.google.com/?hl=zh-tw) 來進行部分練習。

4. 一顆不怕犯錯的心，遇到問題時，請多善用 ChatGPT。
