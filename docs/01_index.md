## 這本書跟其他教學有什麼不同？

在這個 AI 已經無所不在的年代，你需要的不是「學好所有 Python 語法」，而是「知道怎麼問 AI、怎麼表達需求」。  
  
這本小書嚴格的遵循 80/20 法則：

- 只教你「非學不可」的 Python 開發核心知識
- 其餘的部分交給 AI 解決
